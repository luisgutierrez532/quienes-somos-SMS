NUESTRO EQUIPO

Desarrollado por estudiantes para mejorar el sistema de existencias producos existentes . estos productos llegan de diversos proveedores y es necesario tener un regiostro optimo de estos.

# QUE QUEREMOS LOGRAR

Aplicar distintos metodos para para conocer a fondo la cantidad de artículos que se utilizan en la institución para los estudiantes Analizar la información recolectada para solucionar la falta de conocimiento en el inventario de la institución Desarrollar un sistema de información en base a la necesidad de un inventario en base a un sistema gratuito para el uso exclusivo de la institución!

GESTOR DE INVENTARIO

Desarrollo de una base de datos para la administracion de inventarios en la institucion pampurí optimizada para su uso continuo. es un software creado directamente para la institución san Ricardo pampuri, Nuestro punto de enfoque es el sistema gestor de existencias de los artículos de la institución educativa.
